# MultiView ICA

Install
---------

Create virual environment

`virtualenv -p python3 venv`


Activate virtual environment

`source venv/bin/activate`

Move into MultiViewICA directory

``cd multiviewica``

Install MultiViewICA

`pip install -e .`
